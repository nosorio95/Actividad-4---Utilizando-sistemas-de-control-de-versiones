# Proyecto Calculadora

Este proyecto implementa una calculadora básica con operaciones de suma, resta, multiplicación y división.

## Contenido
- Código fuente en `/src`
- Pruebas unitarias en `/tests`
- Historias de usuario en `historias_usuario.md`
- Documentación del proyecto en `/docs`

## Cómo ejecutar

1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta el archivo principal con:

```bash
python main.py
