# Historias de Usuario - Proyecto Calculadora

## HU-01: Sumar números
**Como** usuario  
**Quiero** ingresar dos números y obtener su suma  
**Para** poder realizar cálculos básicos rápidamente.

### Criterios de aceptación
- Debo poder ingresar dos números.
- Al ejecutar la operación, debe mostrarse el resultado correcto.
- Si ingreso valores no numéricos, debe aparecer un mensaje de error.

---

## HU-02: Restar números
**Como** usuario  
**Quiero** ingresar dos números y obtener la resta  
**Para** calcular la diferencia de manera sencilla.

### Criterios de aceptación
- Se deben ingresar dos números.
- El sistema debe mostrar el resultado correcto.
- Si se ingresa un valor inválido, debe notificarse al usuario.

---

## HU-03: Multiplicar números
**Como** usuario  
**Quiero** multiplicar dos números  
**Para** obtener productos sin hacer el cálculo manualmente.

### Criterios de aceptación
- El usuario puede ingresar dos números.
- Debe mostrarse el resultado correcto.
- Ante valores inválidos, el sistema debe mostrar un aviso.

---

## HU-04: Dividir números
**Como** usuario  
**Quiero** dividir un número entre otro  
**Para** obtener un resultado de división de forma rápida.

### Criterios de aceptación
- Debo ingresar dos números.
- Si el divisor es cero, el sistema debe mostrar un mensaje de error.
- El resultado debe calcularse y mostrarse correctamente.

---

## HU-05: Seleccionar operación
**Como** usuario  
**Quiero** un menú donde pueda escoger la operación (sumar, restar, multiplicar, dividir)  
**Para** usar la calculadora de forma intuitiva.

### Criterios de aceptación
- Debe mostrarse un menú con las opciones.
- La opción elegida debe ejecutarse correctamente.
- Si el usuario ingresa una opción inválida, debe mostrarse un mensaje de error.
